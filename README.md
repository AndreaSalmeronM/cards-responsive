# Cards template responsive

A responsive user dashboard built with HTML, CSS, and Bootstrap, showcasing three main sections:

- User Profile Card – Displays personal information with a circular avatar and quick action buttons.
- Balance Card – Styled as a credit card with front and back views, including custom design elements.
- Transaction History – A responsive table listing transaction records with categories, amounts, and comments.

## Features
- Responsive layout using Bootstrap grid system.
- Custom card designs with gradients, chip, magnetic strip, and signature strip.
- FontAwesome icons for buttons and actions.
- Scrollable transaction table with sticky headers.
- Clean separation of HTML and CSS for easy maintenance.
